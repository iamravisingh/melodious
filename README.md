## Prerequisites

- Node.js >= 20.19.4
- Bun >= 1.0
- iOS Simulator (Mac) or Android Studio

## Setup

```bash
# Install dependencies
bun install

# Start development server
bun start

# Run on specific platform
bun run ios
bun run android
bun run web
```
